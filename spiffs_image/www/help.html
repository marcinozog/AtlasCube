👋 WELCOME
AtlasCube is a Wi-Fi internet radio. Stream stations, play music from an SD card or a Bluetooth source, and schedule playback — all from this web interface. The quick steps below link straight to the matching screen.
🌐 Full documentation online
1 · GET ONLINE
Connect AtlasCube to your Wi-Fi so it can stream, then set the clock.
🌐 Wi-Fi 🕐 Time (NTP)
2 · LISTEN TO RADIO
Pick a station, adjust volume and hit Play on the home screen. Manage your station list and shape the sound with the equalizer.
📻 Home 📝 Stations 🎚️ Equalizer
3 · OTHER SOURCES
Stream from a phone over Bluetooth, or play music files from an SD card.
🔵 Bluetooth 🎵 SD Player
4 · SCHEDULE
The Events page has two tabs. 🗓️ Events holds reminders, birthdays, name days and voice notifications. 🎵 Playback schedules the radio or an SD file (or folder) to start automatically at a set time — daily, weekly or one-off.
🗓 Events
5 · MAKE IT YOURS
Tune the device screen, dim & wake behaviour, screensaver and colour theme. Rearrange the on-screen layout with the Layout Editor.
🖥️ Display 🔅 Dim & Wake 🌀 Screensaver 🎨 Theme 🖼️ Layout Editor
6 · SMART HOME
Connect AtlasCube to your home automation over MQTT — control it from a broker and build custom on-screen widgets that show or trigger your devices.
📨 MQTT 🔧 Widgets
7 · ADVANCED
File manager, file editors, hardware pin setup and firmware updates (OTA) live under Settings → Tools. These are powerful — change them only if you know why.
🛠️ Tools
