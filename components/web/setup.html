Firmware update (OTA)

Upload an application image — atlascube.bin, or AtlasCube-<display>-ota.bin for your display from atlascube.net/flash. It is written to the inactive slot and the device reboots into it; settings and pin map are kept. Works from this built-in page even if the web UI partition is wiped. Don't power off during the upload.

Web UI files

⚠️ The web UI files don't match the firmware — left behind by an app-only OTA update. Re-upload the new web UI files below, and include www_version.txt in the selection to clear this warning. Get the matching files (AtlasCube-www.zip) from the latest GitHub release, then extract and upload them below.

Optional / recovery only — the web UI ships inside the firmware, so a normal flash already includes it. Use this just to patch or restore individual files without reflashing.

Pick the files from spiffs_image/www/ (raw, uncompressed) — get the matching set (AtlasCube-www.zip) from the latest release. The device gzips HTML/CSS/JS on upload. Icons (.ico) are not supported here. Include www_version.txt to clear the "web UI out of date" warning.
